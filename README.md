# 毛孩子记忆馆 · 博主共创 Demo

一个用 Flask 写的、面向宠物博主合作的互动式网页 Demo。
用影像、故事和虚拟陪伴，帮养宠人保存和想象与毛孩子的关系。

> 我们不会说它真的回来了。
> 这是一份基于照片和故事的纪念影像创作。

## 运行方式

```bash
cd pet-memory-demo
pip install flask
python app.py
```

打开浏览器访问：

```
http://127.0.0.1:8503
```

## 页面

| 路径        | 说明                                |
| --------- | --------------------------------- |
| `/`       | 首页（产品介绍 + 4 个玩法 + 套餐展示）           |
| `/play`   | 玩法体验页（选玩法 + 填表单）                  |
| `/preview`| 预览页（自动生成纪念文案 + 宠物来信 + 推荐套餐）       |
| `/garden` | 记忆花园（虚拟陪伴：喂食、摸摸头、陪玩、留言）           |
| `/submit` | 提交素材页（上传照片 + 联系方式 + 套餐选择）         |
| `/success`| 提交成功页                              |
| `/creator`| 博主合作页（流程 + 分成 + 申请表单）             |
| `/admin`  | 后台（订单列表、博主申请、玩法/来源统计）             |

## 数据

- 提交保存到：`data/submissions.json`
- 博主申请保存到：`data/creators.json`
- 宠物照片：`uploads/pets/`
- 主人照片：`uploads/owners/`

## 玩法

1. 我和毛孩子一起变老
2. 再拍一张纪念合照
3. 它写给我的一封信
4. 我的电子毛孩子 / 记忆花园
